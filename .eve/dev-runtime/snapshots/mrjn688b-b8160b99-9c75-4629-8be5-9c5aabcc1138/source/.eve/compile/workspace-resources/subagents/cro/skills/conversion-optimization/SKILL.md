# Conversion Optimization
Adapted from coreyhaines31/marketingskills. Map traffic source, visitor intent, promise, information scent, friction, anxiety, distraction, and next step. Label observations separately from hypotheses. Rank opportunities by evidence, impact, and effort. For tests specify hypothesis, variant, audience, primary metric, guardrail, sample/duration rule, instrumentation, and decision rule. Do not test low-traffic changes that should simply be fixed.
