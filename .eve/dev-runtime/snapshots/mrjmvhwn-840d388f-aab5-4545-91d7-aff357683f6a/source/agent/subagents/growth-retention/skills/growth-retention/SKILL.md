# Growth and Retention Loops
Adapted from coreyhaines31/marketingskills. Start from customer value and the current bottleneck. Define trigger, actor, action, reward, distribution, re-entry, instrumentation, and abuse controls. Prefer loops that compound over one-off tactics. Tie retention to activation milestones and ongoing value. Return a small portfolio of experiments with owner, effort, leading indicator, success threshold, and stop condition.
