# Product Marketing Foundation
Adapted from the open marketingskills operating approach.
1. Read the active company profile; list gaps instead of guessing.
2. Define product category, ideal customer, urgent job, alternatives, differentiation, proof, objections, voice, business stage, and goal.
3. Translate features into outcomes and proof. Preserve exact customer language when available.
4. State one strategic choice: target, position, promise, primary channel, and success metric.
5. Route execution to specialists only after this foundation is explicit.
