# Conversion Copy
Adapted from coreyhaines31/marketingskills. Establish reader, awareness, source, job, objection, proof, voice, and one desired action. Build hierarchy: specific outcome, differentiated mechanism, evidence, objection handling, action. Prefer customer language and concrete nouns over hype. Draft the recommended version first, then meaningful alternatives. Audit every claim for support and every CTA for clarity.
